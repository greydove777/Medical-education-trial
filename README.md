# 🧠 神經科臨床案例 001：急性右側肢體無力與構音障礙

> Advanced Medical Educator System — Clinical Case Simulation (Mode A)
> **受教對象**：醫學生（Medical Student）
> **教學主題**：急性神經學症狀的評估邏輯、時間軸思維、NIHSS、中風血管定位

---

## 📂 檔案結構

```
neuro-case-001-acute-stroke/
├── README.md                  ← 你正在看的這份檔案
├── index.md                   ← GitHub Pages 首頁
├── _config.yml                ← GitHub Pages 主題設定
├── simulator.html             ← 獨立運作的互動個案模擬器（對話泡泡+翻牌卡+分支決策）
├── case.md                    ← 案例題目（病史、生命徵象、NIHSS 檢查發現）
├── case-branching.md          ← 進階：治療決策分支劇情（依選擇走向不同結局）
└── answer-key.md              ← 教學解答（NIHSS 計分、血管定位、鑑別診斷、處置、Debriefing）
```

## 📖 使用方式

1. 先閱讀 `case.md`，**不要先看 `answer-key.md`**
2. 依照案例中的引導式提問，自行寫下你的：
   - NIHSS 評分（逐項計分）
   - 血管供應區域定位推論
   - 前 3 個鑑別診斷
   - 治療決策（含時間窗考量）
3. 完成後再對照 `answer-key.md`，檢視思路落差
4. 建議搭配計時器練習：從「病史閱讀」到「下達治療決策」控制在 **15 分鐘內**，模擬急診實境的時間壓力

## 🎯 學習目標 (Learning Objectives)

- [ ] 能正確執行並計算 NIHSS 評分
- [ ] 能根據局部神經學徵候，推論病灶之血管供應區域（vascular territory）
- [ ] 理解「Last known well」在急性中風處置中的關鍵意義
- [ ] 能列出急性局部神經學症狀的合理鑑別診斷，而非僅鎖定中風
- [ ] 熟悉 rt-PA 給藥時間窗與排除條件的基本概念
